Software

Meerkat

A software for reconstructing reciprocal space of the single crystal. The reconstruction is similar to the unwrap function of the Crysalis or similar software, except Meerkat reconstruct the whole three dimensional volume instead of two dimensional planes. Meerkat is a descendent of Xcavate software and contains a number of improvements. Firstly it allows to make reconstructions in crystallographic coordinates, secondly, due to the optimised algorithms the reconstructions are performed faster.

The program is distributed as a python library. Information about the installation the source code can be found on github page.

Yell

Yell is a program for single crystal diffuse scattering refinement using the 3D-ΔPDF merhod. It allows to model all types of short range order correlations and was already successfully applied to organic, inorganic and complex intermetallic compounds. The software features

  • fast FFT based algorithm for diffuse scattering calculation
  • flexible constraint system
  • refinement of the average strucutre and short range order parameters

Since the approach works much faster than direct modelling techniques like Monte Carlo or Molecular Dynamics, we recommend to perform 3D-ΔPDF modelling before using the heavy artillery like DISCUS or ZODS.
